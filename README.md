## About

This is my personal portfolio website built from scratch using pure HTML, CSS, and JavaScript — no frameworks, no templates. It showcases my projects, skills, certifications, and contact information!!.

## Features

- Animated particle network background (Canvas API)
- Custom neon cursor with gold hover effect
- Smooth scroll reveal animations on all sections
- Fully responsive — mobile, tablet, and desktop
- Mobile hamburger menu
- Skill bars with scroll-triggered animation
- Hero stats count-up animation
- Dark cyberpunk neon theme
- Downloadable CV / resume
